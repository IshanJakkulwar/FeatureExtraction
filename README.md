**SUMMARY**
Osteoporosis is a gradually worsening skeletal condition defined by a reduction in bone tissue mass and a subsequent elevated vulnerability to broken bones. Existing methods for diagnosis, including dual-energy X-ray absorptiometry (DXA), face constraints related to expense and availability, which impede the process of identifying the disease early.
We hypothesised that osteoporosis produces radiographically detectable textural changes in trabecular-rich bones, such as the spine, but not in cortical-rich bones, such as the knee. To test this hypothesis, we used a deep learning model (EfficientNet-B0) as a tool to extract texture features from radiographs of normal and osteoporotic knees and spines, followed by principal component analysis (PCA) and statistical testing. In spine radiographs (n = 38), principal component 1 (PC1) separated normal and osteoporotic groups (mean PC1: –0.703 vs 1.968), with a trend toward significance (p = 0.104) and moderate effect size (Cohen’s d = 0.647). In knee radiographs (n = 239), no significant differences were found between groups (mean PC1: –0.199 vs 0.035, p = 0.775, Cohen’s d = 0.047). These findings support our hypothesis that osteoporosis-related texture changes are detectable in trabecular-rich spine bone but not in cortical-rich knee bone. This suggests that radiographic texture analysis could improve osteoporosis screening by targeting trabecular-dominant anatomical sites.
